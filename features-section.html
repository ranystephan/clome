Features

Not another
Electron wrapper.

Every layer — terminal, editor, renderer — is native code talking directly to your hardware. No JavaScript runtime. No web views.

01
Metal

GPU Terminal

Full terminal emulation via libghostty with Metal rendering. Sub-millisecond latency from keystroke to pixel.

02
CoreText

Code Editor

CoreText rendering, rope-backed buffer, LSP completions and diagnostics, multi-cursor, find & replace with regex.

03
LSP

30+ Languages

Syntax highlighting and LSP support for Swift, Rust, Python, TypeScript, Go, Zig, C++, LaTeX, and more.

04
Git

File Explorer

Project tree with live git status, inline rename, brand-colored file type icons, and drag-drop support.

05
Jupyter

Notebooks

Native Jupyter .ipynb rendering and execution. Markdown cells, code cells, image outputs. No browser needed.

06
SQLite

Workspaces

Binary split panes, multi-tab surfaces, session persistence via SQLite. Your layout survives restarts.
