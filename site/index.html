§ 1.0 · Abstract

Abstract

Developer tools until now assumed a single human typing code into an editor. That model breaks when agents read your codebase, run commands, and edit files at the same time you do.

Clome Ecosystem is a small, open-source toolkit built for that reality. Three components (Clome, the desktop environment; Flow, a planning surface inside it; and Clome Flow, a companion app) share a single workspace model. The project is in alpha. Phases land in the repository as numbered increments. This page describes what exists and what is next.
