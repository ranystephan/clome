What's inside

Every layer is native code
talking to your hardware

01

GPU Terminal

Full terminal emulation powered by libghostty with Metal rendering. Sub-millisecond latency from keystroke to pixel.

02

Code Editor

CoreText rendering with a rope-backed buffer. LSP completions, diagnostics, go-to-definition, and multi-cursor editing.

03

30+ Languages

Syntax highlighting and language server support for Swift, Rust, Python, TypeScript, Go, Zig, C++, LaTeX, and more.

04

File Explorer

Project tree with real-time git status indicators, inline rename, brand-colored file icons, and drag-drop support.

05

Notebooks & LaTeX

Native Jupyter notebook rendering with kernel execution. LaTeX compilation to PDF. No browser required.

06

Workspaces

Binary split panes, multi-tab surfaces, persistent sessions via SQLite. Your layout survives every restart.

Architecture

Native all the way down

Every pixel is rendered by AppKit and Metal. The editor uses a rope for O(log n) text operations. The terminal runs through libghostty's Zig-compiled engine.

No 500MB RAM baseline. No web runtime. A macOS app that respects your machine.
